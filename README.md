# 아진산업 연애 시뮬레이션

직장인 인턴 '아진'으로 플레이하는 10일짜리 사내 연애 시뮬레이션 게임.
정규직 전환을 목표로, 4명의 남주 중 한 명과 엔딩을 맞이한다.

## 기술 스택

- **Ren'Py** — 비주얼 노벨 엔진 (스크립트, UI, 세이브 관리)
- **Python** — 게임 로직 (스탯 계산, 엔딩 분기, 아이템 효과)

## 게임 구조

- 10일 × 하루 2행동
- 행동 종류: 업무 / 외모관리 / 휴식 / 상점 / 남주 대화
- 스탯: 매력 / 지능 / 스트레스 / 골드
- 남주 4명 × 호감도 시스템
- 총 11개 엔딩 (해피 4 / 노멀 4 / 특수 1 / 배드 2)

## 실행 방법

1. [Ren'Py SDK](https://www.renpy.org/) 설치
2. 프로젝트 폴더를 Ren'Py 런처에서 열기
3. Launch Project 클릭
